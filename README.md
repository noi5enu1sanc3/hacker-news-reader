## Run Locally

Clone the project

```bash
  git clone git@github.com:noi5enu1sanc3/hacker-news-reader.git
```

Go to the project directory

```bash
  cd hacker-news-reader
```

Install dependencies

```bash
  yarn install
```

Start the server

```bash
  yarn start
```
